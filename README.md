# Gemini SAT PDF Exporter

Google Gemini의 SAT 문제 풀이 및 해설 인터페이스에서 문제와 정답 데이터를 추출하여 PDF로 변환하는 Chrome Extension입니다.

## 기능

- 📚 **Reading 섹션 추출**: 읽기 문제, 지문, 선택지, 정답 및 해설 추출
- 🔢 **Math 섹션 추출**: 수학 문제, 선택지(또는 Grid-in), 정답 및 해설 추출
- 📄 **PDF 생성**: SAT 시험지 형식의 깔끔한 PDF 생성
- 🎨 **직관적인 UI**: Gemini 인터페이스에 "Export to PDF" 버튼 자동 삽입
- ⚡ **실시간 추출**: 현재 화면에 표시된 모든 문제 자동 수집

## 설치 방법

1. 이 저장소를 클론하거나 다운로드합니다.
2. Chrome 브라우저에서 `chrome://extensions/`로 이동합니다.
3. 우측 상단의 "개발자 모드"를 활성화합니다.
4. "압축해제된 확장 프로그램을 로드합니다"를 클릭합니다.
5. 이 프로젝트 폴더를 선택합니다.

## 사용 방법

1. [Google Gemini](https://gemini.google.com)에서 SAT 연습 문제 페이지를 엽니다.
2. 페이지 상단 우측에 "Export to PDF" 버튼이 자동으로 표시됩니다.
3. 버튼을 클릭하면 현재 화면의 모든 SAT 문제가 추출되어 PDF로 다운로드됩니다.

## 기술 스택

- **Manifest V3**: 최신 Chrome Extension 표준
- **JavaScript (ES6+)**: 메인 로직
- **jsPDF**: PDF 생성 라이브러리 (CDN)
- **DOM API**: 데이터 추출
- **MutationObserver**: 동적 콘텐츠 감지

## 주요 기능 설명

### 데이터 추출

- DOM 탐색을 통해 문제 본문, 선택지, 정답, 해설을 자동으로 추출합니다.
- Reading과 Math 섹션을 자동으로 구분하여 처리합니다.
- 동적으로 로딩되는 콘텐츠도 MutationObserver를 통해 감지합니다.

### PDF 생성

- SAT 시험지 형식의 전문적인 레이아웃으로 PDF를 생성합니다.
- 문제 본문은 볼드체로, 선택지는 세로로 나열됩니다.
- 페이지 하단에 "Generated by Our Service (Powered by Gemini)" 문구가 포함됩니다.
- 여러 페이지에 걸친 긴 콘텐츠도 자동으로 페이지 분할합니다.

## 문제 해결

### 문제를 찾을 수 없다는 메시지가 표시되는 경우

Gemini 인터페이스의 HTML 구조가 변경되었을 수 있습니다. 다음을 확인해주세요:

1. F12 개발자 도구를 열어 페이지의 HTML 구조를 확인합니다.
2. 문제가 표시되는 요소의 클래스명이나 ID를 확인합니다.
3. `content.js`의 선택자 패턴을 실제 구조에 맞게 수정합니다.

### PDF가 생성되지 않는 경우

1. 브라우저 콘솔(F12)에서 오류 메시지를 확인합니다.
2. 인터넷 연결을 확인합니다 (jsPDF 라이브러리 로드 필요).
3. Chrome Extension 권한이 올바르게 설정되어 있는지 확인합니다.

## 개발자 참고사항

### HTML 선택자 커스터마이징

`content.js`의 다음 함수들을 수정하여 실제 Gemini 인터페이스 구조에 맞게 조정할 수 있습니다:

- `extractReadingSection()`: Reading 섹션 추출 로직
- `extractMathSection()`: Math 섹션 추출 로직
- `extractChoices()`: 선택지 추출 로직
- `extractAnswer()`: 정답 추출 로직
- `extractExplanation()`: 해설 추출 로직

### 아이콘 추가

`manifest.json`에 아이콘 경로가 지정되어 있습니다. 다음 크기의 아이콘 파일을 추가하세요:
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

아이콘 파일이 없어도 확장 프로그램은 정상 작동합니다.
